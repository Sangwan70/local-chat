## [LocalChat SDK Demo App](https://github.com/frgarciames/localchat-react)

The LocalChat SDK demo app is a robust starting point for developers looking to integrate and customize LocalChat in their applications. Leveraging modern technologies like Tailwind, shadcn/ui, and Biomejs, it provides a smooth development experience and a highly customizable user interface. Refer to the [repository](https://github.com/frgarciames/localchat-react) for more details and to get started.

**Tech Stack:**

- **Tailwind:** A utility-first CSS framework for rapid UI development.
- **shadcn/ui:** A set of high-quality, customizable UI components.
- **LocalChat Web SDK:** The core SDK for interacting with LocalChat.
- **Biomejs formatter/linter:** A tool for maintaining code quality and consistency.
